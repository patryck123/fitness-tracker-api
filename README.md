# Fitness Tracker API

Rastreamento de treinos e exercícios com registro por tipo, duração, calorias queimadas e histórico por período. Ideal para apps de saúde e academia.

## Tecnologias
Java 17 · Spring Boot 3.2 · Spring Data JPA · MySQL · Maven · Swagger/OpenAPI

## Funcionalidades
- Registro de treinos por tipo (Força, Cardio, HIIT, Yoga etc.)
- Exercícios por treino com séries, repetições e peso
- Histórico por período com total de calorias queimadas
- Filtros por tipo de treino e usuário

## Como Executar
```bash
mvn spring-boot:run
# Acesse: http://localhost:8099/swagger-ui.html
```
